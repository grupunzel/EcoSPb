♻️ ЭкоСПб — эко-помощник для жителей Санкт-Петербурга

Telegram-бот для помощи в раздельном сборе отходов.  
Подсказывает, как утилизировать предметы по тексту или фото, показывает карту пунктов приёма, принимает жалобы на переполненные контейнеры.


## 🚀 Функционал

Эко-помощник умеет:
`♻️ Как утилизировать?`: AI (GigaChat) отвечает на текстовый запрос с вопросом о правильной утилизации отходов
`📸 Определить по фото`: YOLO + GigaChat распознают предмет и дают совет
`🗺 Ближайший пункт`: Показывает карту с точками приёма отходов по району
`📢 Отправить жалобу`: Принимает адрес + фото, сохраняет в `complaints.txt`


## 🗂️ Открытые данные

- [ФККО (справочник отходов)](https://rpn.gov.ru/fkko/)
- [Портал контейнеров СПб](https://emtstko.kpoos.gov.spb.ru)
- Яндекс.Карты для визуализации точек приёма


## 🛠️ Технологии

- Python 3.13
- Aiogram 3 (Telegram Bot API)
- GigaChat API (текстовые ответы)
- Ultralytics YOLOv8 (распознавание объектов)
- TXT (хранение жалоб)


## 📁 Структура проекта
├── main.py # Точка входа
├── ai_handler.py # Логика GigaChat
├── complaint_handler.py # Обработчик для жалоб
├── container_handler.py # Обработчик адресов пунктов сбора отходов
├── main_handler.py # Основной обработчик
├── photo_detection_handler.py # Обработчик функции определения по фото
├── utilization_handler.py # Обработчик для текстового запроса
├── states.py # Состояния FSM
├── logs # Папка с логами
├── complaints.txt # Файл с жалобами (создаётся автоматически)
└── requirements.txt # Зависимости


## 🧪 Запуск

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt```

2. Добавьте токен бота и credentials в config/settings.py:

    ```BOT_TOKEN = "ваш_токен"
    GIGA_CREDENTIALS = "Basic ..."```

3. Запустите бота:

    ```python main.py```
