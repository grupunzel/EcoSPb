### ♻️ ЭкоСПб — эко-помощник для жителей Санкт-Петербурга

Telegram-бот для помощи в раздельном сборе отходов.  
Подсказывает, как утилизировать предметы по тексту или фото, показывает карту пунктов приёма, принимает жалобы на переполненные контейнеры.


## 🚀 Функционал

Эко-помощник умеет:<br>
`♻️ Как утилизировать?`: AI (GigaChat) отвечает на текстовый запрос с вопросом о правильной утилизации отходов<br>
`📸 Определить по фото`: YOLO + GigaChat распознают предмет и дают совет<br>
`🗺 Ближайший пункт`: Показывает карту с точками приёма отходов по району<br>
`📢 Отправить жалобу`: Принимает адрес + фото, сохраняет в `complaints.txt`<br>


## 🗂️ Открытые данные

- [ФККО (справочник отходов)](https://rpn.gov.ru/fkko/)
- [Портал контейнеров СПб](https://emtstko.kpoos.gov.spb.ru)
- Яндекс.Карты для визуализации точек приёма


## 🛠️ Технологии

- Python 3.13
- Aiogram 3 (Telegram Bot API)
- GigaChat API (текстовые ответы)
- Ultralytics YOLOv8 (распознавание объектов)
- TXT (хранение жалоб)
